# devops-eng-53
Information sharing repository for devops-eng-53 class

Prerequisites

1. Register your GitHub individual account.
2. Send your GitHub user id and registered email address to yzou@spartaglobal.com.

Using Teams and Github repository

1. Prepare for each lesson in advance as instructedin the Teams Engineering 53 group.
2. Download lab start code from the Teams Engineering 53 group for lab excises.
3. Create your private GitHub repository in which you will complete your lab exercises.
4. Give yuan-zou-github access to your GitHub lab repository so that the trainer is able to check your lab exercises.
5. Discuss lesson content and Q & A for general or specific issues in the Teams Engineering 53 group.
6. Receive homework assignments in the Teams Engineering 53 group, and complete them in your own GitHub repository.

## Prerequisites

### For trainee

1. Uninstall software in Windows
	* Vagrant
	* VirtualBox
	* ChefDK
	* Ruby

2. Install software
	* GitBash for windows (as system administrator)
	* Atom

3. Setup a private GitHub account
	* Send link to your Github repository to Yuan Zou in Teams chat or email yzou@spartaglobal.com

4. Create GitHub repositories for labs
	* Working code
	* Complimenting README to explain what has been completed for each lesson. This is for readiness of client engagement and interviewing.

5. Familiarise yourself with GitHub
	* Create private repository
	* Give your classmates and trainer access to you repository
	* Clone GitHub repository to local for modification
	* Commit local changes to your private GitHub repository
	* Push your local files to a new private GitHub repository

6. Register a private account at https://www.docker.com

7. Prepare for questions
	* What is DevOps?
	* Why do you choose to be a DevOps engineer?
	* Why do you choose Sparta Global?
	* What benefits does DevOps bring to IT and business?
	* What qualities make a good DevOps engineer?
	* What have you learned so far in Sparta Global Academy?
